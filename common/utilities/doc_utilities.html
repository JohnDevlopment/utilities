Description

This package contains a series of handy utility functions that may aid in the development of Tcl-based applications.

Public API

assert expression ?message?

If expression evaluates to false the current program exits with a non-zero status, indicating an error. message is printed to standard error as this happens; if it is omitted, it defaults to the standard "assertion failed" message.

To use this function, you need to set to 0. This is done automatically if the DEBUG enviroment variable is set to a non-zero value.

bitset intVar bit flag

Set or clear the given bit in intVar; set if flag is true, clear otherwise.

const name value

Creates a new variable by the name name and sets its value to value. At the time this function is called, name cannot already exist. Returns value.

This function creates a readonly variable: once its value is set upon creation, it cannot be changed again. Any attempt to do so will result in an error.

deref var

Returns the value of the variable var.

do body while cond

Implements a do-while loop. Evaluates the script body once and then continues to do so while cond evaluates to true.

lambda args body

Returns an apply lambda function with the arguments args and body as the function body.

lremove listVar begin ?end?

Removes one or more elements from a list found in the variable listVar. begin and end are index values that specify the start and end of a range of elements to remove. Indices are interpreted in the same manner as lreplace.

Returns an empty string.

pincr var ?i?

Implements a post-fix increment operator in Tcl. Increments the numeric value in var by the amount specified by i. If i is not provided, then 1 is used as the default.

The return value is the value of var prior to the increment.

popFront listVar

Removes the first element of the list contained in listVar and returns it. If the list is empty, so is the return value.

settemp var ?value?

Creates a temporary value that deletes itself at idle time. Internally, var gets added to a queue that is processed at the start of the next event loop, using after idle.

Randon Number Generation

random integer ?min? max

Returns a random integer between min and max. If min is omitted, it defaults to zero.

random string length

Returns a randomly generated string of length characters. The string will contain characters between a-z (upper and lowercase) and an underscore.

Option Processing

These functions belong to the ::Options namespace.

getOptions specs optVar argvVar

Processes commandline options according to the rules defined in specs.

specs is a list of lists containing recognized options, where each element takes the form of {option default}. If the option doesn't contain any special suffixes, the option is treated as a boolean flag: it is set to true if provided, and false otherwise. If option ends in ".arg", the option expects an argument. default is used as the argument in the case that such option is not provided.

The options in argvVar are put into the array variable optVar, where each index is an option and its value is the corresponding argument.
